# Universal DIY Autonomy Package Experience

A single-page, futuristic website presenting the Universal DIY Autonomy Package. The page highlights the modular sensor, AI, and connectivity stack that can retrofit scooters, bicycles, and small EVs into smart autonomous vehicles.

## Preview

Open `index.html` in your browser to explore the interactive concept experience. The site features animated constellations, glassmorphism UI, and responsive layouts that scale from desktops to mobile devices.

## Structure

- `index.html` – Page markup and content sections
- `styles.css` – Visual design, animations, and responsive rules
- `script.js` – Canvas constellation background, reveal animations, and module selector interactions

## Customization

Update the copy within each section to align with your product messaging. You can also tweak the module configurations, metrics, and milestones to reflect roadmap updates.

## Deployment

The project is framework-free, so any static host will work:

```bash
# Example using Python
python -m http.server 8000
```

Navigate to `http://localhost:8000` and open `index.html` to view the site locally.
